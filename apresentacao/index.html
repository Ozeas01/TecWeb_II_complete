WebLearning Games

Uma plataforma interativa e gamificada para desafios de programação

Apresentado por: Ozeas do Carmo Carneiro

O que é o projeto?


  • Uma plataforma web gamificada voltada para o ensino de tecnologias web.
  • Torna o aprendizado prático e envolvente.
  • Desafios interativos de HTML, CSS e JavaScript.
  • Foco em uma experiência de usuário (UX/UI) premium, inspirada em grandes empresas.

Tecnologias Usadas

  • React & TypeScript: Componentes interativos, seguros e sem erros.
  • Vite: Ferramenta de build para máxima velocidade.
  • Tailwind CSS: Estilização responsiva, limpa e moderna.
  • Framer Motion: Biblioteca para animações fluidas e micro-interações.
  • Vercel: Hospedagem e deploy contínuo em nuvem.

Por que escolhemos isso?

  • Alta Performance: Funcionamento sem recarregamento (SPA).
  • Manutenibilidade: Código limpo e seguro usando tipagem do TypeScript.
  • Design "Uau": Tailwind + Framer Motion permitiram criar uma estética moderna de mercado, substituindo visuais estáticos por interfaces vivas.

Aplicações Futuras

  • Backend: Sistema de login, salvamento no banco de dados e Ranking Global.
  • Novas Trilhas: React Avançado, Node.js, Python.
  • Acessibilidade (a11y): Melhorar o uso para leitores de tela e teclado.
  • Mobile (PWA): Transformar em um app instalável no celular.
  • Multiplayer: Batalhas de código ao vivo!

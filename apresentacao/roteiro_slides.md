# Slide 2: O que é o WebLearning Games?
- **Conceito:** Uma plataforma web gamificada voltada para o ensino e prática de tecnologias web.
- **Objetivo:** Tornar o aprendizado de código mais envolvente, prático e interativo.
- **Diferenciais:**
  - Desafios práticos de HTML, CSS e JavaScript.
  - Feedback visual em tempo real.
  - Interface inspirada em produtos digitais de alta tecnologia e padrão corporativo (Premium UX/UI).

---

# Slide 3: Tecnologias Usadas
- **React & TypeScript:** Para a construção dos componentes interativos com tipagem estática que previne erros no código.
- **Vite:** Ferramenta de build que garante um carregamento e desenvolvimento extremamente rápidos.
- **Tailwind CSS:** Framework de estilos que permite uma estilização responsiva, limpa e moderna sem sair do código HTML/JSX.
- **Framer Motion:** Biblioteca responsável por todas as animações fluidas, efeitos de deslize e micro-interações dos botões.
- **Vercel:** Utilizada para hospedagem e deploy contínuo, colocando o site no ar de forma rápida e segura.

---

# Slide 4: Por que essas tecnologias?
- **Performance:** Vite + React entregam um site super veloz (Single Page Application - SPA), sem recarregamentos travados de página.
- **Manutenibilidade:** O uso do TypeScript garante um código muito mais seguro e fácil de dar manutenção por outros desenvolvedores.
- **Design:** O casamento entre Tailwind CSS e Framer Motion permitiu chegar em uma estética moderna, substituindo visuais estáticos por interfaces "vivas" que reagem ao usuário.

# Slide 5: Aplicações Futuras e Melhorias
- **Backend & Banco de Dados:** Implementar sistema de contas de usuário (Login), banco de dados para salvar progresso e um Ranking Global (Leaderboard).
- **Novas Trilhas de Conhecimento:** Expandir os jogos para além do frontend básico, adicionando trilhas de React avançado, Node.js e Python.
- **Acessibilidade Aprimorada (a11y):** Garantir que 100% da plataforma possa ser usada por tecnologias assistivas (leitores de tela) e navegação pura por teclado.
- **PWA (Progressive Web App):** Transformar o site em um aplicativo instalável nos smartphones, permitindo resolução de desafios offline.
- **Modo Multiplayer:** Batalhas de código em tempo real entre alunos ou turmas.

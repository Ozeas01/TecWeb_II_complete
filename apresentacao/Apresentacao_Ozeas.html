Apresentação de Projeto

WebLearning Games

Uma plataforma interativa, gamificada e moderna para desafios reais de programação web.

Desenvolvido e Apresentado por

Ozeas do Carmo Carneiro

O que é o WebLearning Games?

  • Plataforma educacional focada no ensino e prática imersiva de tecnologias de desenvolvimento web.
  • Traz o conceito de gamificação, transformando estudos teóricos e monótonos em desafios interativos.
  • Foca nas três linguagens pilares: HTML, CSS e JavaScript.
  • Interface (UI/UX) projetada com estética premium e responsiva, fugindo do visual padrão de sites acadêmicos.

Stack de Tecnologias

  • React + TypeScript: Base do projeto. Componentização robusta e tipagem estática que elimina erros.
  • Vite: Ferramenta de build de nova geração (extremamente rápida).
  • Vercel: Hospedagem na nuvem com integração e deploy contínuos (CI/CD).
  • Tailwind CSS: Estilização ágil e responsiva usando classes utilitárias no próprio código.
  • Framer Motion: Biblioteca profissional de animação (responsável por transições e micro-interações fluidas na interface).

O Porquê das Escolhas

  • Performance Superior (SPA): O site não recarrega a página ao navegar. A experiência é fluida como um aplicativo nativo.
  • Código Seguro e Limpo: O TypeScript atua como uma barreira de segurança durante o desenvolvimento.
  • Estética de Mercado: O uso conjunto do Tailwind CSS e Framer Motion cria um visual que se equipara a produtos de grandes empresas (Apple, Vercel, Nubank).

Visão de Futuro

  • Backend & Banco de Dados: Implementar autenticação de usuários e um Ranking Global de pontos.
  • Batalhas Multiplayer: Desafios de código simultâneos entre alunos da mesma turma.
  • Expansão de Conteúdo: Criação de novas trilhas para React Avançado, Node.js e Python.
  • Acessibilidade Total: Suporte completo para leitores de tela e navegação por teclado (a11y).
